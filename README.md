# Koi Scripts
Scripts for QuantumultX,JSBox,NodeJs

本仓库使用[Peng-YM](https://github.com/Peng-YM)的[OpenAPI](https://github.com/Peng-YM/QuanX/tree/master/Tools/OpenAPI)，因此理论上部分QuantumultX脚本支持Surge、Loon，但重写配置需自行适配
